# HR System Perfect Screenshot Extension

이 확장프로그램은 HR 시스템의 팝업을 완벽하게 캡처하기 위해 특별히 제작되었습니다.

## 🚀 설치 방법

### Chrome/Edge 브라우저:

1. **Chrome 개발자 모드 활성화**
   - Chrome에서 `chrome://extensions/` 주소창에 입력
   - 우상단의 "개발자 모드" 토글 켜기

2. **확장프로그램 로드**
   - "압축해제된 확장 프로그램을 로드합니다" 클릭
   - `C:\hr-system\public\extension` 폴더 선택
   - "폴더 선택" 클릭

3. **확장프로그램 고정**
   - 브라우저 우상단 퍼즐 아이콘 클릭
   - "HR System Perfect Screenshot" 찾기
   - 📌 고정 버튼 클릭

## 🎯 사용 방법

### 방법 1: 자동 감지 캡처
1. HR 시스템 페이지에서 팝업 열기
2. 확장프로그램 아이콘 클릭
3. 자동으로 팝업 감지하여 완벽 캡처

### 방법 2: 수동 선택 캡처
1. 확장프로그램 아이콘 클릭
2. 팝업 창에서 원하는 캡처 방식 선택:
   - 📸 팝업 완전 캡처
   - 📄 전체 페이지 캡처

## ✨ 특징

- ✅ **스크롤 내용 완전 캡처**: 팝업의 모든 내용 포함
- ✅ **고화질 이미지**: 2배 스케일로 선명한 결과
- ✅ **자동 팝업 감지**: HR 시스템 팝업 자동 인식
- ✅ **완벽한 레이아웃**: 차트, 표, 색상 모두 보존
- ✅ **즉시 다운로드**: 캡처 완료 시 자동 저장

## 🔧 지원 팝업

- 목표달성률 상세 팝업
- 워라밸 지표 상세 팝업
- 기타 HR 시스템 모든 팝업

## 🛠️ 문제 해결

만약 캡처가 되지 않는다면:

1. **페이지 새로고침** 후 다시 시도
2. **팝업이 완전히 로드**될 때까지 기다린 후 캡처
3. **브라우저 콘솔(F12)** 에서 오류 메시지 확인
4. 확장프로그램을 **비활성화했다가 다시 활성화**

## 🎉 완벽한 캡처 보장!

이 확장프로그램은 기존 html2canvas의 한계를 극복하고, 모든 스크롤 제약을 해제하여 팝업의 전체 내용을 완벽하게 캡처합니다.
